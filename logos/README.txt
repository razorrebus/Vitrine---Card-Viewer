Team logos for the card back.

Name each file after the team, lowercased with spaces as underscores, .png:
  Seattle Mariners     -> seattle_mariners.png
  New York Yankees     -> new_york_yankees.png
  St. Louis Cardinals  -> st_louis_cardinals.png   (punctuation becomes _)

Square images work best (they're shown in a 62x62 box, scaled to fit).
If no matching file exists for a card's Team, the logo area is left blank.
